## 🎨 Design System

### Color Palette
```
Primary Green:   #10b981  (Logo, buttons, highlights)
Dark Green:      #059669  (Hover states)
Light Green:     #d1fae5  (Backgrounds, accents)
Dark Text:       #111827  (Main content)
Medium Grey:     #6b7280  (Secondary text)
Light Grey:      #f9fafb  (Backgrounds)
White:           #ffffff  (Cards, containers)
```

### Typography
- Font: System fonts (Apple, Windows, fallback to sans-serif)
- Headings: Bold (700 weight)
- Body: Regular (400 weight)
- Labels: Medium/Semi-bold (500-600 weight)

### Spacing
- Consistent 8px base grid
- Multiples: 16px, 24px, 32px, 48px
- Form padding: 14-16px
- Card padding: 24-32px
- Button padding: 12-16px vertical

### Component Design
- Border radius: 12-16px (modern, friendly)
- Shadows: Subtle elevation (2-8px blur)
- Hover lift: -2px to -4px translateY
- Transitions: 0.3s ease for smooth feel
